Welcome to online code review service.
Why pay for github/gitlab if you can make the same yourself?

Our idea is very simple:
  • Clone the repository.
  • Allow reviewer to add comments in the code.
  • Push the comments to a new branch (we're not doing this while in beta, but it's just one more command).
  • The developer can get the branch and work on the code according with the comment.


At the same time we're also testing our simple CI service. We build and test the code via `mvn test` periodically.
